# 3D Snake Game

一个基于 **Three.js** 的 3D 贪吃蛇小游戏。

## 功能
- 🍎 食物发光（自发光 + 点光源脉冲）
- 🎮 鼠标控制视角（拖拽旋转、滚轮缩放）
- 🎵 背景音乐（内置 WebAudio，可开关）
- 🏆 分数排行榜（本地存储 Top 5 展示）
- 💀 游戏结束界面（本局得分 + 排名）
- 📱 手机触控方向键

## 操作
- `W/A/S/D` 或方向键：控制蛇移动
- `Space`：开始/重开
- 鼠标：旋转/缩放视角
- 右下角：手机触控按钮

## 本地运行
```bash
python3 -m http.server 8000
```
打开 `http://localhost:8000` 即可。
